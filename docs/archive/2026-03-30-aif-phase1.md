# AIF Phase 1: Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the AIF parser, semantic IR, and compilers (HTML, Markdown, LLM-view) plus a Markdown importer and CLI.

**Architecture:** Cargo workspace with crates: `aif-core` (IR types), `aif-parser` (logos lexer + hand-written recursive descent), `aif-html` (HTML emitter), `aif-markdown` (MD emitter + importer), `aif-lml` (LLM-view emitter), `aif-cli` (binary). Two-phase parsing: block-level (line-oriented) then inline-level (delimiter matching). Persistent AST for multi-backend compilation.

**Tech Stack:** Rust, logos (lexer), clap (CLI), serde/serde_json (IR serialization), insta (snapshot tests), pretty_assertions, html-escape

use logos::Logos;

#[derive(Logos, Debug, Clone, PartialEq)]
#[logos(skip r"[ \t]+")]  // skip horizontal whitespace between tokens on a line
pub enum Token {
    // Document metadata: #key: value
    #[regex(r"#[a-zA-Z_][a-zA-Z0-9_]*:")]
    MetaKey,

    // Block directive: @blocktype
    #[regex(r"@[a-zA-Z_][a-zA-Z0-9_]*")]
    BlockDirective,

    // Attribute open bracket
    #[token("[")]
    BracketOpen,

    // Attribute close bracket
    #[token("]")]
    BracketClose,

    // Colon (used after block directive for title)
    #[token(":")]
    Colon,

    // Equals sign in attributes
    #[token("=")]
    Equals,

    // Comma in attributes
    #[token(",")]
    Comma,

    // Code fence (3+ backticks)
    #[regex(r"```[`]*")]
    CodeFence,

    // Block quote marker
    #[token(">")]
    BlockQuoteMarker,

    // Unordered list marker
    #[token("-")]
    Dash,

    // Ordered list marker: digits followed by dot
    #[regex(r"[0-9]+\.")]
    OrderedMarker,

    // Thematic break (3+ dashes alone on a line, handled at block level)
    #[regex(r"---[-]*")]
    ThematicBreak,

    // Newline
    #[token("\n")]
    Newline,

    // Inline emphasis markers
    #[token("**")]
    DoubleStar,

    #[token("*")]
    Star,

    // Inline code
    #[token("`")]
    Backtick,

    // Link/image markers
    #[token("[")]
    // already have BracketOpen

    #[token("](")]
    LinkMiddle,

    #[token(")")]
    Paren,

    // Reference: @ref[id]
    #[token("@ref")]
    RefMarker,

    // Footnote markers
    #[token("~")]
    Tilde,

    // Any other text content
    #[regex(r"[^\n\[\]()@#`*~>=,\\-]+")]
    Text,

    // Escape sequence
    #[regex(r"\\.", priority = 10)]
    Escaped,
}

/// A token with its span in the source
#[derive(Debug, Clone, PartialEq)]
pub struct SpannedToken {
    pub token: Token,
    pub span: std::ops::Range<usize>,
    pub slice: String,
}

pub fn lex(input: &str) -> Vec<SpannedToken> {
    let mut lexer = Token::lexer(input);
    let mut tokens = Vec::new();
    while let Some(result) = lexer.next() {
        if let Ok(token) = result {
            tokens.push(SpannedToken {
                token,
                span: lexer.span(),
                slice: lexer.slice().to_string(),
            });
        }
    }
    tokens
}

use aif_core::ast::Attrs;

/// Parse attribute string like "id=intro, src=data.csv" or just "intro" (bare id)
pub fn parse_attrs(input: &str) -> Attrs {
    let input = input.trim();
    if input.is_empty() {
        return Attrs::new();
    }

    let mut attrs = Attrs::new();
    let parts: Vec<&str> = input.split(',').map(|s| s.trim()).collect();

    for (i, part) in parts.iter().enumerate() {
        if let Some((key, value)) = part.split_once('=') {
            let key = key.trim();
            let value = value.trim();
            if key == "id" {
                attrs.id = Some(value.to_string());
            } else {
                attrs.pairs.insert(key.to_string(), value.to_string());
            }
        } else if i == 0 && !part.contains('=') {
            // First bare value is treated as id
            attrs.id = Some(part.to_string());
        }
    }

    attrs
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn parse_empty_attrs() {
        let attrs = parse_attrs("");
        assert_eq!(attrs.id, None);
        assert!(attrs.pairs.is_empty());
    }

    #[test]
    fn parse_bare_id() {
        let attrs = parse_attrs("intro");
        assert_eq!(attrs.id, Some("intro".to_string()));
    }

    #[test]
    fn parse_key_value_attrs() {
        let attrs = parse_attrs("id=main, src=data.csv");

- [ ] **Step 3: Implement block parser**

The block parser works line-by-line on the raw input (not on lexer tokens — document parsers work better with line-oriented parsing for block level). We use the lexer only for inline content.

In `crates/aif-parser/src/block.rs`:
```rust
use aif_core::ast::*;
use aif_core::error::ParseError;
use aif_core::span::Span;
use crate::attrs::parse_attrs;
use crate::inline;

pub struct BlockParser<'a> {
    input: &'a str,
    lines: Vec<(usize, &'a str)>, // (byte offset, line content)
    pos: usize, // current line index
}

impl<'a> BlockParser<'a> {
    pub fn new(input: &'a str) -> Self {
        let mut offset = 0;
        let lines: Vec<(usize, &str)> = input
            .split('\n')
            .map(|line| {
                let start = offset;
                offset += line.len() + 1; // +1 for the \n
                (start, line)
            })
            .collect();
        Self { input, lines, pos: 0 }
    }

    pub fn parse(&mut self) -> Result<Document, Vec<ParseError>> {
        let mut doc = Document::new();
        let mut errors = Vec::new();

        // Parse metadata lines at start
        while self.pos < self.lines.len() {
            let (offset, line) = self.lines[self.pos];
            if let Some(rest) = line.strip_prefix('#') {
                if let Some((key, value)) = rest.split_once(':') {
                    let key = key.trim().to_string();
                    let value = value.trim().to_string();
                    doc.metadata.insert(key, value);
                    self.pos += 1;
                    continue;
                }
            }
            break;
        }

        // Skip blank lines after metadata
        self.skip_blank_lines();

        // Parse blocks
        while self.pos < self.lines.len() {
            self.skip_blank_lines();
            if self.pos >= self.lines.len() {
                break;
            }

            match self.parse_block() {
                Ok(Some(block)) => doc.blocks.push(block),
                Ok(None) => {} // blank line or skip
                Err(e) => {
                    errors.push(e);
                    self.pos += 1; // skip problematic line
                }
            }
        }

        if errors.is_empty() {
            Ok(doc)
        } else {
            Err(errors)
        }
    }

    fn parse_block(&mut self) -> Result<Option<Block>, ParseError> {
        let (offset, line) = self.lines[self.pos];

        if line.trim().is_empty() {
            self.pos += 1;
            return Ok(None);
        }

        // Block directive: @type[attrs]: title
        if line.starts_with('@') {
            return self.parse_block_directive();
        }

        // Code fence
        if line.starts_with("```") {
            return self.parse_code_fence();
        }

        // Block quote
        if line.starts_with('>') {
            return self.parse_block_quote();
        }

        // Thematic break: --- or more
        if line.starts_with("---") && line.trim().chars().all(|c| c == '-') {
            let span = Span::new(offset, offset + line.len());
            self.pos += 1;
            return Ok(Some(Block { kind: BlockKind::ThematicBreak, span }));
        }

        // Unordered list
        if line.starts_with("- ") {
            return self.parse_unordered_list();
        }

        // Ordered list
        if line.chars().next().map(|c| c.is_ascii_digit()).unwrap_or(false)
            && line.contains(". ")
        {
            if let Some(dot_pos) = line.find(". ") {
                let prefix = &line[..dot_pos];
                if prefix.chars().all(|c| c.is_ascii_digit()) {
                    return self.parse_ordered_list();
                }
            }
        }

        // Default: paragraph
        self.parse_paragraph()
    }

    fn parse_block_directive(&mut self) -> Result<Option<Block>, ParseError> {
        let (offset, line) = self.lines[self.pos];

        // Parse: @type[attrs]: title
        // or:    @type[attrs]
        // or:    @type: title
        // or:    @type
        let rest = &line[1..]; // skip @

        // Extract block type name
        let type_end = rest.find(|c: char| !c.is_alphanumeric() && c != '_')
            .unwrap_or(rest.len());
        let block_type = &rest[..type_end];
        let mut remaining = &rest[type_end..];

        // Parse optional [attrs]
        let attrs = if remaining.starts_with('[') {
            if let Some(close) = remaining.find(']') {
                let attr_str = &remaining[1..close];
                remaining = &remaining[close + 1..];
                parse_attrs(attr_str)
            } else {
                Attrs::new()
            }
        } else {
            Attrs::new()
        };

        // Parse optional : title
        let title_text = if remaining.starts_with(':') {
            let t = remaining[1..].trim();
            if t.is_empty() { None } else { Some(t.to_string()) }
        } else {
            None
        };

        self.pos += 1;

        // Collect body lines (non-blank, non-directive lines following)
        let body = self.collect_body_lines();

        let span = Span::new(offset, self.current_offset());

        // Dispatch based on block type
        match block_type {
            "section" => {
                let title = title_text.map(|t| inline::parse_inline(&t)).unwrap_or_default();
                // Parse body as child blocks
                let mut child_parser = BlockParser::new(&body);
                let child_doc = child_parser.parse().map_err(|e| e.into_iter().next().unwrap())?;
                Ok(Some(Block {
                    kind: BlockKind::Section {
                        attrs,
                        title,
                        children: child_doc.blocks,
                    },
                    span,
                }))
            }
            "claim" | "evidence" | "definition" | "theorem" | "assumption"
            | "result" | "conclusion" | "requirement" | "recommendation" => {
                let semantic_type = match block_type {
                    "claim" => SemanticBlockType::Claim,
                    "evidence" => SemanticBlockType::Evidence,
                    "definition" => SemanticBlockType::Definition,
                    "theorem" => SemanticBlockType::Theorem,
                    "assumption" => SemanticBlockType::Assumption,
                    "result" => SemanticBlockType::Result,
                    "conclusion" => SemanticBlockType::Conclusion,
                    "requirement" => SemanticBlockType::Requirement,
                    "recommendation" => SemanticBlockType::Recommendation,
                    _ => unreachable!(),
                };
                let title = title_text.map(|t| inline::parse_inline(&t));
                let content = inline::parse_inline(&body);
                Ok(Some(Block {
                    kind: BlockKind::SemanticBlock {
                        block_type: semantic_type,
                        attrs,
                        title,
                        content,
                    },
                    span,
                }))
            }
            "callout" => {
                let callout_type = match attrs.get("type").unwrap_or("note") {
                    "warning" => CalloutType::Warning,
                    "info" => CalloutType::Info,
                    "tip" => CalloutType::Tip,
                    _ => CalloutType::Note,
                };
                let content = inline::parse_inline(&body);
                Ok(Some(Block {
                    kind: BlockKind::Callout {
                        callout_type,
                        attrs,
                        content,
                    },
                    span,
                }))
            }
            "table" => {
                let caption_line = self.extract_caption_from_body_lines(&body);
                let caption = caption_line.map(|c| inline::parse_inline(&c));
                Ok(Some(Block {
                    kind: BlockKind::Table {
                        attrs,
                        caption,
                        headers: Vec::new(),
                        rows: Vec::new(),
                    },
                    span,
                }))
            }
            "figure" => {
                let src = attrs.get("src").unwrap_or("").to_string();
                let caption_line = self.extract_caption_from_body_lines(&body);
                let caption = caption_line.map(|c| inline::parse_inline(&c));
                Ok(Some(Block {
                    kind: BlockKind::Figure {
                        attrs,
                        caption,
                        src,
                    },
                    span,
                }))
            }
            "code" => {
                let lang = attrs.get("lang").map(|s| s.to_string());
                Ok(Some(Block {
                    kind: BlockKind::CodeBlock {
                        lang,
                        attrs,
                        code: body,
                    },
                    span,
                }))
            }
            _ => {
                // Unknown directive — treat as semantic block with generic type
                let content = inline::parse_inline(&body);
                Ok(Some(Block {
                    kind: BlockKind::SemanticBlock {
                        block_type: SemanticBlockType::Claim, // fallback
                        attrs,
                        title: title_text.map(|t| inline::parse_inline(&t)),
                        content,
                    },
                    span,
                }))
            }
        }
    }

    fn parse_code_fence(&mut self) -> Result<Option<Block>, ParseError> {
        let (offset, line) = self.lines[self.pos];
        let fence = line.trim();
        let lang = fence.strip_prefix("```").and_then(|rest| {
            let l = rest.trim();
            if l.is_empty() { None } else { Some(l.to_string()) }
        });

        self.pos += 1;
        let mut code_lines = Vec::new();

        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            if line.trim().starts_with("```") {
                self.pos += 1;
                break;
            }
            code_lines.push(line);
            self.pos += 1;
        }

        let code = code_lines.join("\n");
        let span = Span::new(offset, self.current_offset());

        Ok(Some(Block {
            kind: BlockKind::CodeBlock {
                lang,
                attrs: Attrs::new(),
                code,
            },
            span,
        }))
    }

    fn parse_block_quote(&mut self) -> Result<Option<Block>, ParseError> {
        let offset = self.lines[self.pos].0;
        let mut quote_lines = Vec::new();

        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            if let Some(rest) = line.strip_prefix('>') {
                quote_lines.push(rest.strip_prefix(' ').unwrap_or(rest));
                self.pos += 1;
            } else {
                break;
            }
        }

        let quote_text = quote_lines.join("\n");
        let mut inner_parser = BlockParser::new(&quote_text);
        let inner_doc = inner_parser.parse().map_err(|e| e.into_iter().next().unwrap())?;
        let span = Span::new(offset, self.current_offset());

        Ok(Some(Block {
            kind: BlockKind::BlockQuote {
                content: inner_doc.blocks,
            },
            span,
        }))
    }

    fn parse_unordered_list(&mut self) -> Result<Option<Block>, ParseError> {
        let offset = self.lines[self.pos].0;
        let mut items = Vec::new();

        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            if let Some(rest) = line.strip_prefix("- ") {
                let content = inline::parse_inline(rest);
                items.push(ListItem {
                    content,
                    children: Vec::new(),
                });
                self.pos += 1;
            } else if line.starts_with("  - ") {
                // Nested list item — add as child of last item
                if let Some(last) = items.last_mut() {
                    let rest = &line[4..];
                    let content = inline::parse_inline(rest);
                    last.children.push(Block {
                        kind: BlockKind::List {
                            ordered: false,
                            items: vec![ListItem { content, children: Vec::new() }],
                        },
                        span: Span::new(self.lines[self.pos].0, self.lines[self.pos].0 + line.len()),
                    });
                }
                self.pos += 1;
            } else {
                break;
            }
        }

        let span = Span::new(offset, self.current_offset());
        Ok(Some(Block {
            kind: BlockKind::List { ordered: false, items },
            span,
        }))
    }

    fn parse_ordered_list(&mut self) -> Result<Option<Block>, ParseError> {
        let offset = self.lines[self.pos].0;
        let mut items = Vec::new();

        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            if let Some(dot_pos) = line.find(". ") {
                let prefix = &line[..dot_pos];
                if prefix.chars().all(|c| c.is_ascii_digit()) {
                    let rest = &line[dot_pos + 2..];
                    let content = inline::parse_inline(rest);
                    items.push(ListItem {
                        content,
                        children: Vec::new(),
                    });
                    self.pos += 1;
                    continue;
                }
            }
            break;
        }

        let span = Span::new(offset, self.current_offset());
        Ok(Some(Block {
            kind: BlockKind::List { ordered: true, items },
            span,
        }))
    }

    fn parse_paragraph(&mut self) -> Result<Option<Block>, ParseError> {
        let offset = self.lines[self.pos].0;
        let mut para_lines = Vec::new();

        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            if line.trim().is_empty() || line.starts_with('@') || line.starts_with('#')
                || line.starts_with("```") || line.starts_with('>') || line.starts_with("- ")
            {
                break;
            }
            // Also break on ordered list start
            if line.chars().next().map(|c| c.is_ascii_digit()).unwrap_or(false) {
                if let Some(dot_pos) = line.find(". ") {
                    if line[..dot_pos].chars().all(|c| c.is_ascii_digit()) {
                        break;
                    }
                }
            }
            para_lines.push(line);
            self.pos += 1;
        }

        if para_lines.is_empty() {
            return Ok(None);
        }

        let text = para_lines.join(" ");
        let content = inline::parse_inline(&text);
        let span = Span::new(offset, self.current_offset());

        Ok(Some(Block {
            kind: BlockKind::Paragraph { content },
            span,
        }))
    }

    fn skip_blank_lines(&mut self) {
        while self.pos < self.lines.len() && self.lines[self.pos].1.trim().is_empty() {
            self.pos += 1;
        }
    }

    fn current_offset(&self) -> usize {
        if self.pos < self.lines.len() {
            self.lines[self.pos].0
        } else {
            self.input.len()
        }
    }

    fn collect_body_lines(&mut self) -> String {
        let mut body_lines = Vec::new();
        while self.pos < self.lines.len() {
            let (_, line) = self.lines[self.pos];
            // Stop at blank line, new directive, or metadata
            if line.trim().is_empty() || line.starts_with('@') || line.starts_with('#') {
                break;
            }
            // Stop at caption: prefix — we'll still include it in the body string
            // for extract_caption_from_body_lines
            body_lines.push(line);
            self.pos += 1;
        }
        body_lines.join("\n")
    }

    fn extract_caption_from_body_lines(&self, body: &str) -> Option<String> {
        for line in body.lines() {
            if let Some(rest) = line.strip_prefix("caption:") {
                return Some(rest.trim().to_string());
            }
        }
        None
    }
}

    pos: usize,

impl<'a> InlineParser<'a> {
    fn new(input: &'a str) -> Self {
        Self { input, pos: 0 }
    }

    fn parse(&mut self) -> Vec<Inline> {
        let mut inlines = Vec::new();
        let mut text_start = self.pos;

        while self.pos < self.input.len() {
            let remaining = &self.input[self.pos..];

            // **bold**
            if remaining.starts_with("**") {
                self.flush_text(text_start, &mut inlines);
                if let Some(content) = self.parse_delimited("**", "**") {
                    let inner = InlineParser::new(&content).parse();
                    inlines.push(Inline::Strong { content: inner });
                    text_start = self.pos;
                    continue;
                }
                // Not closed — treat as text
                self.pos += 2;
                continue;
            }

            if remaining.starts_with('*') && !remaining.starts_with("**") {
                self.flush_text(text_start, &mut inlines);
                if let Some(content) = self.parse_delimited("*", "*") {
                    let inner = InlineParser::new(&content).parse();
                    inlines.push(Inline::Emphasis { content: inner });
                    text_start = self.pos;
                    continue;
                }
                self.pos += 1;
                continue;
            }

            // `code`
            if remaining.starts_with('`') {
                self.flush_text(text_start, &mut inlines);
                if let Some(code) = self.parse_delimited("`", "`") {
                    inlines.push(Inline::InlineCode { code });
                    text_start = self.pos;
                    continue;
                }
                self.pos += 1;
                continue;

            if remaining.starts_with('[') {
                self.flush_text(text_start, &mut inlines);
                if let Some((text, url)) = self.parse_link() {
                    let text_inlines = InlineParser::new(&text).parse();
                    inlines.push(Inline::Link { text: text_inlines, url });
                    text_start = self.pos;
                    continue;
                }
                self.pos += 1;
                continue;
            }

            // @ref[id]
            if remaining.starts_with("@ref[") {
                self.flush_text(text_start, &mut inlines);
                if let Some(target) = self.parse_ref() {
                    inlines.push(Inline::Reference { target });
                    text_start = self.pos;
                    continue;
                }
                self.pos += 5;
                continue;
            }

            // ~footnote~
            if remaining.starts_with('~') {
                self.flush_text(text_start, &mut inlines);
                if let Some(content) = self.parse_delimited("~", "~") {
                    let inner = InlineParser::new(&content).parse();
                    inlines.push(Inline::Footnote { content: inner });
                    text_start = self.pos;
                    continue;
                }
                self.pos += 1;
                continue;
            }

            // Escape: \*
            if remaining.starts_with('\\') && self.pos + 1 < self.input.len() {
                self.flush_text(text_start, &mut inlines);
                self.pos += 1; // skip backslash
                text_start = self.pos;
                self.pos += 1; // include escaped char as text
                continue;
            }

            self.pos += remaining.chars().next().map(|c| c.len_utf8()).unwrap_or(1);
        }

        self.flush_text(text_start, &mut inlines);
        inlines
    }

    fn flush_text(&self, start: usize, inlines: &mut Vec<Inline>) {
        if start < self.pos {
            let text = self.input[start..self.pos].to_string();
            if !text.is_empty() {
                inlines.push(Inline::Text { text });
            }
        }
    }

    fn parse_delimited(&mut self, open: &str, close: &str) -> Option<String> {
        let start = self.pos + open.len();
        if start >= self.input.len() {
            return None;
        }
        let remaining = &self.input[start..];
        if let Some(end) = remaining.find(close) {
            let content = remaining[..end].to_string();
            self.pos = start + end + close.len();
            Some(content)
        } else {
            None
        }
    }

    fn parse_link(&mut self) -> Option<(String, String)> {
        // [text](url)
        let start = self.pos + 1; // skip [
        let remaining = &self.input[start..];
        let close_bracket = remaining.find(']')?;
        let text = remaining[..close_bracket].to_string();
        let after = &remaining[close_bracket + 1..];
        if !after.starts_with('(') {
            return None;
        }
        let url_start = close_bracket + 2; // skip ](
        let url_remaining = &remaining[url_start..];
        let close_paren = url_remaining.find(')')?;
        let url = url_remaining[..close_paren].to_string();
        self.pos = start + url_start + close_paren + 1;
        Some((text, url))
    }

    fn parse_ref(&mut self) -> Option<String> {
        // @ref[id]
        let start = self.pos + 5; // skip @ref[
        let remaining = &self.input[start..];
        let close = remaining.find(']')?;
        let target = remaining[..close].to_string();
        self.pos = start + close + 1;
        Some(target)
    }
}

use std::fmt::Write;

pub fn emit_html(doc: &Document) -> String {
    let mut out = String::new();
    writeln!(out, "<!DOCTYPE html>").unwrap();
    writeln!(out, "<html>").unwrap();
    writeln!(out, "<head>").unwrap();
    if let Some(title) = doc.metadata.get("title") {
        writeln!(out, "<title>{}</title>", escape_html(title)).unwrap();
    }
    if let Some(summary) = doc.metadata.get("summary") {
        writeln!(out, "<meta name=\"description\" content=\"{}\">", escape_html(summary)).unwrap();
    }
    writeln!(out, "</head>").unwrap();
    writeln!(out, "<body>").unwrap();

    if let Some(title) = doc.metadata.get("title") {
        writeln!(out, "<h1>{}</h1>", escape_html(title)).unwrap();
    }

    for block in &doc.blocks {
        emit_block(&mut out, block, 2);
    }

    writeln!(out, "</body>").unwrap();
    writeln!(out, "</html>").unwrap();
    out
}

fn emit_block(out: &mut String, block: &Block, heading_level: u8) {
    match &block.kind {
        BlockKind::Section { attrs, title, children } => {
            let id_attr = attrs.id.as_ref().map(|id| format!(" id=\"{}\"", escape_html(id))).unwrap_or_default();
            writeln!(out, "<section{}>", id_attr).unwrap();
            let level = heading_level.min(6);
            write!(out, "<h{}>", level).unwrap();
            emit_inlines(out, title);
            writeln!(out, "</h{}>", level).unwrap();
            for child in children {
                emit_block(out, child, heading_level + 1);
            }
            writeln!(out, "</section>").unwrap();
        }
        BlockKind::Paragraph { content } => {
            write!(out, "<p>").unwrap();
            emit_inlines(out, content);
            writeln!(out, "</p>").unwrap();
        }
        BlockKind::SemanticBlock { block_type, attrs, title, content } => {
            let type_name = format!("{:?}", block_type).to_lowercase();
            let id_attr = attrs.id.as_ref().map(|id| format!(" id=\"{}\"", escape_html(id))).unwrap_or_default();
            writeln!(out, "<div class=\"aif-{}\"{}>" , type_name, id_attr).unwrap();
            if let Some(title) = title {
                write!(out, "<strong>").unwrap();
                emit_inlines(out, title);
                writeln!(out, "</strong>").unwrap();
            }
            write!(out, "<p>").unwrap();
            emit_inlines(out, content);
            writeln!(out, "</p>").unwrap();
            writeln!(out, "</div>").unwrap();
        }
        BlockKind::Callout { callout_type, content, .. } => {
            let class = format!("{:?}", callout_type).to_lowercase();
            writeln!(out, "<aside class=\"aif-callout aif-{}\">", class).unwrap();
            write!(out, "<p>").unwrap();
            emit_inlines(out, content);
            writeln!(out, "</p>").unwrap();
            writeln!(out, "</aside>").unwrap();
        }
        BlockKind::Table { attrs, caption, headers, rows } => {
            let id_attr = attrs.id.as_ref().map(|id| format!(" id=\"{}\"", escape_html(id))).unwrap_or_default();
            writeln!(out, "<table{}>", id_attr).unwrap();
            if let Some(caption) = caption {
                write!(out, "<caption>").unwrap();
                emit_inlines(out, caption);
                writeln!(out, "</caption>").unwrap();
            }
            if !headers.is_empty() {
                writeln!(out, "<thead><tr>").unwrap();
                for header in headers {
                    write!(out, "<th>").unwrap();
                    emit_inlines(out, header);
                    write!(out, "</th>").unwrap();
                }
                writeln!(out, "</tr></thead>").unwrap();
            }
            if !rows.is_empty() {
                writeln!(out, "<tbody>").unwrap();
                for row in rows {
                    write!(out, "<tr>").unwrap();
                    for cell in row {
                        write!(out, "<td>").unwrap();
                        emit_inlines(out, cell);
                        write!(out, "</td>").unwrap();
                    }
                    writeln!(out, "</tr>").unwrap();
                }
                writeln!(out, "</tbody>").unwrap();
            }
            writeln!(out, "</table>").unwrap();
        }
        BlockKind::Figure { attrs, caption, src } => {
            let id_attr = attrs.id.as_ref().map(|id| format!(" id=\"{}\"", escape_html(id))).unwrap_or_default();
            writeln!(out, "<figure{}>", id_attr).unwrap();
            writeln!(out, "<img src=\"{}\" alt=\"\">", escape_html(src)).unwrap();
            if let Some(caption) = caption {
                write!(out, "<figcaption>").unwrap();
                emit_inlines(out, caption);
                writeln!(out, "</figcaption>").unwrap();
            }
            writeln!(out, "</figure>").unwrap();
        }
        BlockKind::CodeBlock { lang, code, .. } => {
            let lang_class = lang.as_ref().map(|l| format!(" class=\"language-{}\"", escape_html(l))).unwrap_or_default();
            writeln!(out, "<pre><code{}>", lang_class).unwrap();
            write!(out, "{}", escape_html(code)).unwrap();
            writeln!(out, "</code></pre>").unwrap();
        }
        BlockKind::BlockQuote { content } => {
            writeln!(out, "<blockquote>").unwrap();
            for child in content {
                emit_block(out, child, heading_level);
            }
            writeln!(out, "</blockquote>").unwrap();
        }
        BlockKind::List { ordered, items } => {
            let tag = if *ordered { "ol" } else { "ul" };
            writeln!(out, "<{}>", tag).unwrap();
            for item in items {
                write!(out, "<li>").unwrap();
                emit_inlines(out, &item.content);
                for child in &item.children {
                    emit_block(out, child, heading_level);
                }
                writeln!(out, "</li>").unwrap();
            }
            writeln!(out, "</{}>", tag).unwrap();
        }
        BlockKind::ThematicBreak => {
            writeln!(out, "<hr>").unwrap();
        }
    }
}

fn emit_inlines(out: &mut String, inlines: &[Inline]) {
    for inline in inlines {
        match inline {
            Inline::Text { text } => write!(out, "{}", escape_html(text)).unwrap(),
            Inline::Strong { content } => {
                write!(out, "<strong>").unwrap();
                emit_inlines(out, content);
                write!(out, "</strong>").unwrap();
            }
            Inline::Emphasis { content } => {
                write!(out, "<em>").unwrap();
                emit_inlines(out, content);
                write!(out, "</em>").unwrap();
            }
            Inline::InlineCode { code } => {
                write!(out, "<code>{}</code>", escape_html(code)).unwrap();
            }
            Inline::Link { text, url } => {
                write!(out, "<a href=\"{}\">", escape_html(url)).unwrap();
                emit_inlines(out, text);
                write!(out, "</a>").unwrap();
            }
            Inline::Reference { target } => {
                write!(out, "<a href=\"#{}\" class=\"aif-ref\">{}</a>", escape_html(target), escape_html(target)).unwrap();
            }
            Inline::Footnote { content } => {
                write!(out, "<span class=\"aif-footnote\">").unwrap();
                emit_inlines(out, content);
                write!(out, "</span>").unwrap();
            }
            Inline::SoftBreak => write!(out, "\n").unwrap(),
            Inline::HardBreak => write!(out, "<br>").unwrap(),
        }
    }
}

fn escape_html(s: &str) -> String {
    s.replace('&', "&amp;")
        .replace('<', "&lt;")
        .replace('>', "&gt;")
        .replace('"', "&quot;")
}

use aif_core::ast::*;
use std::fmt::Write;

pub fn emit_markdown(doc: &Document) -> String {
    let mut out = String::new();

    if let Some(title) = doc.metadata.get("title") {
        writeln!(out, "# {}", title).unwrap();
        writeln!(out).unwrap();
    }

    for block in &doc.blocks {
        emit_block(&mut out, block, 2);
        writeln!(out).unwrap();
    }

    out.trim_end().to_string() + "\n"
}

fn emit_block(out: &mut String, block: &Block, heading_level: u8) {
    match &block.kind {
        BlockKind::Section { title, children, .. } => {
            let level = heading_level.min(6) as usize;
            let hashes: String = "#".repeat(level);
            write!(out, "{} ", hashes).unwrap();
            emit_inlines(out, title);
            writeln!(out).unwrap();
            writeln!(out).unwrap();
            for child in children {
                emit_block(out, child, heading_level + 1);
                writeln!(out).unwrap();
            }
        }
        BlockKind::Paragraph { content } => {
            emit_inlines(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::SemanticBlock { block_type, title, content, .. } => {
            let label = format!("{:?}", block_type);
            if let Some(title) = title {
                write!(out, "**{}:** ", label).unwrap();
                emit_inlines(out, title);
                writeln!(out).unwrap();
                writeln!(out).unwrap();
            } else {
                writeln!(out, "**{}:**", label).unwrap();
                writeln!(out).unwrap();
            }
            emit_inlines(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::Callout { callout_type, content, .. } => {
            let label = format!("{:?}", callout_type);
            write!(out, "> **{}:** ", label).unwrap();
            emit_inlines(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::Table { caption, headers, rows, .. } => {
            if let Some(caption) = caption {
                write!(out, "*").unwrap();
                emit_inlines(out, caption);
                writeln!(out, "*").unwrap();
                writeln!(out).unwrap();
            }
            if !headers.is_empty() {
                write!(out, "|").unwrap();
                for header in headers {
                    write!(out, " ").unwrap();
                    emit_inlines(out, header);
                    write!(out, " |").unwrap();
                }
                writeln!(out).unwrap();
                write!(out, "|").unwrap();
                for _ in headers {
                    write!(out, " --- |").unwrap();
                }
                writeln!(out).unwrap();
            }
            for row in rows {
                write!(out, "|").unwrap();
                for cell in row {
                    write!(out, " ").unwrap();
                    emit_inlines(out, cell);
                    write!(out, " |").unwrap();
                }
                writeln!(out).unwrap();
            }
        }
        BlockKind::Figure { src, caption, .. } => {
            writeln!(out, "![{}]({})", caption.as_ref().map(|c| inlines_to_text(c)).unwrap_or_default(), src).unwrap();
        }
        BlockKind::CodeBlock { lang, code, .. } => {
            writeln!(out, "```{}", lang.as_deref().unwrap_or("")).unwrap();
            writeln!(out, "{}", code).unwrap();
            writeln!(out, "```").unwrap();
        }
        BlockKind::BlockQuote { content } => {
            for child in content {
                let mut inner = String::new();
                emit_block(&mut inner, child, heading_level);
                for line in inner.lines() {
                    writeln!(out, "> {}", line).unwrap();
                }
            }
        }
        BlockKind::List { ordered, items } => {
            for (i, item) in items.iter().enumerate() {
                if *ordered {
                    write!(out, "{}. ", i + 1).unwrap();
                } else {
                    write!(out, "- ").unwrap();
                }
                emit_inlines(out, &item.content);
                writeln!(out).unwrap();
                for child in &item.children {
                    let mut inner = String::new();
                    emit_block(&mut inner, child, heading_level);
                    for line in inner.lines() {
                        writeln!(out, "  {}", line).unwrap();
                    }
                }
            }
        }
        BlockKind::ThematicBreak => {
            writeln!(out, "---").unwrap();
        }
    }
}

fn emit_inlines(out: &mut String, inlines: &[Inline]) {
    for inline in inlines {
        match inline {
            Inline::Text { text } => write!(out, "{}", text).unwrap(),
            Inline::Strong { content } => {
                write!(out, "**").unwrap();
                emit_inlines(out, content);
                write!(out, "**").unwrap();
            }
            Inline::Emphasis { content } => {
                write!(out, "*").unwrap();
                emit_inlines(out, content);
                write!(out, "*").unwrap();
            }
            Inline::InlineCode { code } => write!(out, "`{}`", code).unwrap(),
            Inline::Link { text, url } => {
                write!(out, "[").unwrap();
                emit_inlines(out, text);
                write!(out, "]({})", url).unwrap();
            }
            Inline::Reference { target } => write!(out, "[{}](#{})", target, target).unwrap(),
            Inline::Footnote { content } => {
                write!(out, "[^").unwrap();
                emit_inlines(out, content);
                write!(out, "]").unwrap();
            }
            Inline::SoftBreak => writeln!(out).unwrap(),
            Inline::HardBreak => writeln!(out, "  ").unwrap(),
        }
    }
}

fn inlines_to_text(inlines: &[Inline]) -> String {
    let mut s = String::new();
    emit_inlines(&mut s, inlines);
    s
}

use aif_core::ast::*;
use std::fmt::Write;

/// Emit a compact, semantic, low-token LLM-optimized view.
/// Format:
///   [DOC title="..." summary="..."]
///   [SECTION id=intro] Introduction
///   Some text here.
///   [CLAIM id=c1] This is a claim.
///   [/SECTION]
///   [/DOC]
pub fn emit_lml(doc: &Document) -> String {
    let mut out = String::new();

    // Document header
    write!(out, "[DOC").unwrap();
    for (key, value) in &doc.metadata {
        write!(out, " {}=\"{}\"", key, value).unwrap();
    }
    writeln!(out, "]").unwrap();

    for block in &doc.blocks {
        emit_block(&mut out, block);
    }

    writeln!(out, "[/DOC]").unwrap();
    out
}

fn emit_block(out: &mut String, block: &Block) {
    match &block.kind {
        BlockKind::Section { attrs, title, children } => {
            write!(out, "[SECTION").unwrap();
            emit_attrs_compact(out, attrs);
            write!(out, "] ").unwrap();
            emit_inlines_plain(out, title);
            writeln!(out).unwrap();
            for child in children {
                emit_block(out, child);
            }
            writeln!(out, "[/SECTION]").unwrap();
        }
        BlockKind::Paragraph { content } => {
            emit_inlines_plain(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::SemanticBlock { block_type, attrs, title, content } => {
            let tag = format!("{:?}", block_type).to_uppercase();
            write!(out, "[{}", tag).unwrap();
            emit_attrs_compact(out, attrs);
            write!(out, "]").unwrap();
            if let Some(t) = title {
                write!(out, " ").unwrap();
                emit_inlines_plain(out, t);
            }
            writeln!(out).unwrap();
            emit_inlines_plain(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::Callout { callout_type, content, .. } => {
            let tag = format!("{:?}", callout_type).to_uppercase();
            write!(out, "[{}] ", tag).unwrap();
            emit_inlines_plain(out, content);
            writeln!(out).unwrap();
        }
        BlockKind::Table { attrs, caption, .. } => {
            write!(out, "[TABLE").unwrap();
            emit_attrs_compact(out, attrs);
            write!(out, "]").unwrap();
            if let Some(c) = caption {
                write!(out, " ").unwrap();
                emit_inlines_plain(out, c);
            }
            writeln!(out).unwrap();
        }
        BlockKind::Figure { attrs, caption, src } => {
            write!(out, "[FIGURE").unwrap();
            emit_attrs_compact(out, attrs);
            write!(out, " src=\"{}\"]", src).unwrap();
            if let Some(c) = caption {
                write!(out, " ").unwrap();
                emit_inlines_plain(out, c);
            }
            writeln!(out).unwrap();
        }
        BlockKind::CodeBlock { lang, code, .. } => {
            write!(out, "[CODE").unwrap();
            if let Some(l) = lang {
                write!(out, " lang=\"{}\"", l).unwrap();
            }
            writeln!(out, "]").unwrap();
            writeln!(out, "{}", code).unwrap();
            writeln!(out, "[/CODE]").unwrap();
        }
        BlockKind::BlockQuote { content } => {
            writeln!(out, "[QUOTE]").unwrap();
            for child in content {
                emit_block(out, child);
            }
            writeln!(out, "[/QUOTE]").unwrap();
        }
        BlockKind::List { ordered, items } => {
            for (i, item) in items.iter().enumerate() {
                if *ordered {
                    write!(out, "{}. ", i + 1).unwrap();
                } else {
                    write!(out, "- ").unwrap();
                }
                emit_inlines_plain(out, &item.content);
                writeln!(out).unwrap();
            }
        }
        BlockKind::ThematicBreak => {
            writeln!(out, "---").unwrap();
        }
    }
}

fn emit_attrs_compact(out: &mut String, attrs: &Attrs) {
    if let Some(id) = &attrs.id {
        write!(out, " id={}", id).unwrap();
    }
    for (key, value) in &attrs.pairs {
        write!(out, " {}={}", key, value).unwrap();
    }
}

/// Emit inlines as plain text (strip formatting, keep content)
fn emit_inlines_plain(out: &mut String, inlines: &[Inline]) {
    for inline in inlines {
        match inline {
            Inline::Text { text } => write!(out, "{}", text).unwrap(),
            Inline::Strong { content } | Inline::Emphasis { content } | Inline::Footnote { content } => {
                emit_inlines_plain(out, content);
            }
            Inline::InlineCode { code } => write!(out, "{}", code).unwrap(),
            Inline::Link { text, url } => {
                emit_inlines_plain(out, text);
                write!(out, " ({})", url).unwrap();
            }
            Inline::Reference { target } => write!(out, "@{}", target).unwrap(),
            Inline::SoftBreak | Inline::HardBreak => write!(out, " ").unwrap(),
        }
    }
}

use aif_core::ast::*;
use aif_core::span::Span;
use pulldown_cmark::{Event, Parser, Tag, TagEnd, CodeBlockKind};

pub fn import(input: &str) -> Document {
    let parser = Parser::new(input);
    let mut doc = Document::new();
    let mut block_stack: Vec<BlockBuilder> = Vec::new();

    for event in parser {
        match event {
            Event::Start(tag) => {
                block_stack.push(BlockBuilder::new(tag));
            }
            Event::End(tag_end) => {
                if let Some(builder) = block_stack.pop() {
                    let block = builder.finish();
                    if let Some(parent) = block_stack.last_mut() {
                        parent.add_child_block(block);
                    } else if let Some(block) = block {
                        doc.blocks.push(block);
                    }
                }
            }
            Event::Text(text) => {
                if let Some(builder) = block_stack.last_mut() {
                    builder.add_inline(Inline::Text { text: text.to_string() });
                }
            }
            Event::Code(code) => {
                if let Some(builder) = block_stack.last_mut() {
                    builder.add_inline(Inline::InlineCode { code: code.to_string() });
                }
            }
            Event::SoftBreak => {
                if let Some(builder) = block_stack.last_mut() {
                    builder.add_inline(Inline::SoftBreak);
                }
            }
            Event::HardBreak => {
                if let Some(builder) = block_stack.last_mut() {
                    builder.add_inline(Inline::HardBreak);
                }
            }
            Event::Rule => {
                let block = Block {
                    kind: BlockKind::ThematicBreak,
                    span: Span::empty(),
                };
                if let Some(parent) = block_stack.last_mut() {
                    parent.add_child_block(Some(block));
                } else {
                    doc.blocks.push(block);
                }
            }
            _ => {}
        }
    }

    // Extract title from first heading if present
    if let Some(first) = doc.blocks.first() {
        if let BlockKind::Section { title, .. } = &first.kind {
            let title_text: String = title.iter().map(|i| match i {
                Inline::Text { text } => text.clone(),
                _ => String::new(),
            }).collect();
            if !title_text.is_empty() {
                doc.metadata.insert("title".into(), title_text);
            }
        }
    }

    doc
}

struct BlockBuilder {
    tag: Tag<'static>,
    inlines: Vec<Inline>,
    children: Vec<Block>,
    code_text: String,
}

impl BlockBuilder {
    fn new(tag: Tag<'_>) -> Self {
        // Convert to owned/static lifetime
        let tag: Tag<'static> = unsafe { std::mem::transmute(tag) };
        Self {
            tag,
            inlines: Vec::new(),
            children: Vec::new(),
            code_text: String::new(),
        }
    }

    fn add_inline(&mut self, inline: Inline) {
        match &self.tag {
            Tag::CodeBlock(_) => {
                if let Inline::Text { text } = &inline {
                    self.code_text.push_str(text);
                }
            }
            _ => self.inlines.push(inline),
        }
    }

    fn add_child_block(&mut self, block: Option<Block>) {
        if let Some(block) = block {
            self.children.push(block);
        }
    }

    fn finish(self) -> Option<Block> {
        let span = Span::empty();
        match &self.tag {
            Tag::Heading { level, .. } => {
                let level_num = *level as u8;
                if level_num == 1 {
                    // H1 -> Section (top-level)
                    Some(Block {
                        kind: BlockKind::Section {
                            attrs: Attrs::new(),
                            title: self.inlines,
                            children: self.children,
                        },
                        span,
                    })
                } else {
                    Some(Block {
                        kind: BlockKind::Section {
                            attrs: Attrs::new(),
                            title: self.inlines,
                            children: self.children,
                        },
                        span,
                    })
                }
            }
            Tag::Paragraph => {
                Some(Block {
                    kind: BlockKind::Paragraph { content: self.inlines },
                    span,
                })
            }
            Tag::CodeBlock(kind) => {
                let lang = match kind {
                    CodeBlockKind::Fenced(lang) => {
                        let l = lang.to_string();
                        if l.is_empty() { None } else { Some(l) }
                    }
                    CodeBlockKind::Indented => None,
                };
                let code = self.code_text.trim_end_matches('\n').to_string();
                Some(Block {
                    kind: BlockKind::CodeBlock {
                        lang,
                        attrs: Attrs::new(),
                        code,
                    },
                    span,
                })
            }
            Tag::BlockQuote(_) => {
                Some(Block {
                    kind: BlockKind::BlockQuote { content: self.children },
                    span,
                })
            }
            Tag::List(start) => {
                let ordered = start.is_some();
                let items = self.children.into_iter().filter_map(|b| {
                    match b.kind {
                        BlockKind::Paragraph { content } => Some(ListItem {
                            content,
                            children: Vec::new(),
                        }),
                        _ => None,
                    }
                }).collect();
                Some(Block {
                    kind: BlockKind::List { ordered, items },
                    span,
                })
            }
            Tag::Item => {
                // List items become paragraphs, parent List will convert them
                if !self.inlines.is_empty() {
                    Some(Block {
                        kind: BlockKind::Paragraph { content: self.inlines },
                        span,
                    })
                } else if let Some(first) = self.children.into_iter().next() {
                    Some(first)
                } else {
                    None
                }
            }
            Tag::Emphasis => {
                // Inline container — wrap in Emphasis, return as pseudo-block
                // This shouldn't normally create a block; inline handling should catch it
                None
            }
            Tag::Strong => None,
            _ => None,
        }
    }
}
```

Note: The `transmute` for lifetime extension is a known pattern with pulldown-cmark. A cleaner alternative is to collect all events first, but this works for the MVP. We can refactor later.

use clap::{Parser, Subcommand};
use std::fs;
use std::path::PathBuf;

#[derive(Parser)]
#[command(name = "aif")]
#[command(about = "AIF: AI-native Interchange Format compiler")]
#[command(version)]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Compile an AIF document to an output format
    Compile {
        /// Input .aif file
        input: PathBuf,

        /// Output format: html, markdown, lml, json
        #[arg(short, long, default_value = "html")]
        format: String,

        /// Output file (defaults to stdout)
        #[arg(short, long)]
        output: Option<PathBuf>,
    },
    /// Import a Markdown file to AIF IR (JSON)
    Import {
        /// Input Markdown file
        input: PathBuf,

        /// Output file (defaults to stdout)
        #[arg(short, long)]
        output: Option<PathBuf>,
    },
    /// Dump the parsed IR as JSON
    DumpIr {
        /// Input .aif file
        input: PathBuf,
    },
}

fn main() {
    let cli = Cli::parse();

    match cli.command {
        Commands::Compile { input, format, output } => {
            let source = fs::read_to_string(&input).unwrap_or_else(|e| {
                eprintln!("Error reading {}: {}", input.display(), e);
                std::process::exit(1);
            });
            let doc = aif_parser::parse(&source).unwrap_or_else(|errors| {
                for e in &errors {
                    eprintln!("{}", e);
                }
                std::process::exit(1);
            });

            let result = match format.as_str() {
                "html" => aif_html::render_html(&doc),
                "markdown" | "md" => aif_markdown::render_markdown(&doc),
                "lml" => aif_lml::render_lml(&doc),
                "json" => serde_json::to_string_pretty(&doc).unwrap(),
                _ => {
                    eprintln!("Unknown format: {}. Supported: html, markdown, lml, json", format);
                    std::process::exit(1);
                }
            };

            if let Some(output_path) = output {
                fs::write(&output_path, &result).unwrap_or_else(|e| {
                    eprintln!("Error writing {}: {}", output_path.display(), e);
                    std::process::exit(1);
                });
                eprintln!("Wrote {}", output_path.display());
            } else {
                print!("{}", result);
            }
        }
        Commands::Import { input, output } => {
            let source = fs::read_to_string(&input).unwrap_or_else(|e| {
                eprintln!("Error reading {}: {}", input.display(), e);
                std::process::exit(1);
            });
            let doc = aif_markdown::import_markdown(&source);
            let json = serde_json::to_string_pretty(&doc).unwrap();

            if let Some(output_path) = output {
                fs::write(&output_path, &json).unwrap_or_else(|e| {
                    eprintln!("Error writing {}: {}", output_path.display(), e);
                    std::process::exit(1);
                });
                eprintln!("Wrote {}", output_path.display());
            } else {
                print!("{}", json);
            }
        }
        Commands::DumpIr { input } => {
            let source = fs::read_to_string(&input).unwrap_or_else(|e| {
                eprintln!("Error reading {}: {}", input.display(), e);
                std::process::exit(1);
            });
            let doc = aif_parser::parse(&source).unwrap_or_else(|errors| {
                for e in &errors {
                    eprintln!("{}", e);
                }
                std::process::exit(1);
            });
            let json = serde_json::to_string_pretty(&doc).unwrap();
            println!("{}", json);
        }
    }
}
