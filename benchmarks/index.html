Key insight: Plain text extraction (PDF-text 561K, cleaned HTML ~600K) is cheapest but carries zero structure. AIF LML Aggressive (980K) costs ~75% more tokens but is the only format that preserves full semantic types at fewer tokens than raw Markdown (1.26M). The real comparison isn't AIF vs HTML (unfair) — it's AIF vs Markdown (22% cheaper) and AIF vs flat text (~75% more, but with typed blocks).
